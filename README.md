# Health Wellness Blog SEO

Create responsible, SEO-ready health and wellness blog content without unsafe medical promises.

## Purpose

This skill helps agents develop search-intent briefs, SEO titles, meta descriptions, outlines, evidence-aware drafts, FAQs, internal-link ideas, practical wellness tips, safety-friendly wording, disclaimers, responsible CTAs, source plans, claim audits, fact-check checklists, and publication-readiness reviews.

It is designed for health bloggers, wellness brands, coaches, newsletter writers, agencies, and editorial teams that need useful content while respecting medical-safety boundaries.

## Installation

Use this folder as a version-controlled Agent Skill repository. For Codex discovery, place or copy the `health-wellness-blog-seo` folder into your skills directory, such as:

```text
~/.codex/skills/health-wellness-blog-seo
```

## Usage

Invoke the skill for tasks such as:

```text
Use $health-wellness-blog-seo to create a search-intent brief and outline for "how to improve sleep hygiene naturally."
```

```text
Use $health-wellness-blog-seo to audit this supplement article for unsafe claims and rewrite the risky sections.
```

## Outputs

- Search-intent briefs
- SEO titles and meta descriptions
- Article outlines and drafts
- FAQs and schema ideas
- Internal-link ideas
- Source plans
- Claim audits
- Fact-check checklists
- Publication-readiness reviews

## Safety Boundaries

The skill does not diagnose, prescribe, promise cures, replace medical care, guarantee health outcomes, fabricate citations, or create unsafe product claims.
